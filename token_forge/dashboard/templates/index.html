How Prompt Caching Works

Prompt Caching saves up to **90%** of input costs. However, providers charge in blocks (typically 1024 tokens). By padding and structuring your prompts with TokenForge, you:

  • Align prompt prefixes exactly to block multiples to guarantee 100% cache hits.
  • Strip boilerplate dates, headers, and logs so only raw information is billed.
  • Selectively prune tools and system prompts based on current query routing.
